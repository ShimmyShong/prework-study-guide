Prework Study Guide

HTML

  • HTML stands for Hypertext Markup Language, which is one of the three cornerstone languages used in web development.
  • The head element contains information (or metadata) that helps the browser to render the page correctly.
  • The body element contains the content that is shown to the user. All the elements that we want the user to see and interact with must be placed inside this element.
  • When working with visible HTML elements, the content is contained between an opening tag and a closing tag.

CSS

  • Cascading Style Sheets is a programming langauge used by developers to define how webpage content is presented to users.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • External CSS style sheets allows for developers to keep all their CSS rules in a separate file, making design changes easier.

Git

  • git status: check what branch we're in and what files have been modified.
  • git checkout: change what branch we're in.
  • git checkout -b (branch name): creates a new branch and switches to it.
  • git pull origin main: pulls the source code from GitHUb to ensure that we have tha latest changes and commits.
  • git push origin (branch name): this will push your changes to GitHub. Make sure you are pushing to your feature branch, not the main branch.
